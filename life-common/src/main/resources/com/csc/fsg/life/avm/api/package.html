The Allowable Values sub-system is the API used for inquiry only access to the AVM database.

If your application is using the com.csc.fsg.life.context.UserContext classs to manage state you can use the com.csc.fsg.life.avm.AllowableValuesHelper class. If not then you may need to use the API's directly in this package.

The starting point to access all capabilities in this package is the AVMManager class. That class is primed with a configuration bean AVMConfigBean.
